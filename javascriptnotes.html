Section 1 Notes

Console

The console is a panel that displays important messages like errors. If we want to see what the computer is doing with our code, we can print, or log to our console directly.

Console refers to an object, a collection of data and actions, that we can use in our code.

Keywords are words that are built into the JavaScript language, so that the computer can recognize them and treat them specially.

One action that is built into the console object is the .log()method. When we write console.log() what we put inside the parentheses will get printed, or logged to the console.

Always use a semicolon to denote the end of a line. Example: console.log(5); will log 5 to the console.


Comments

Comments only exist for human readers, just as in HTML and CSS.

Comments are typically used to explain what the code is doing, to make it easier to understand the code when coming back to it later.

A comment can be made in JavaScript by using // (For a single line comment), or /* comment */ (For a multi-line comment).


Data Types

Data Types are the classifications we give to the different kinds of data that we use in programming. There are seven fundamental data types.

  1. Number. Any number, including decimals.
  2. String. Any grouping of characters on your keyboard surrounded by single quotes, or double quotes. Single quotes are typically more preferable. String can also be thought of as simply text.
  3. Boolean. There are only two possible values for this data type: true, or false. Can be thought of as on and off switches, or as the answers to a yes or no question.
  4. Null. Represents the intentional absence of a value, and is represented by the keyword null.
  5. Undefined. Denoted by the keyword undefined. It also represents the absence of a value, although it has a different use than null.
  6. Symbol. Unique identifiers, useful in more complex coding.
  7. Object. Collections of related data.

The first six data types are considered primitive data types, they are the most basic data types in JavaScript.


Arithmetic Operators

An operator is a character that performs a task in our code. JavaScript has several built-in arithmetic operators that allow us to perform mathematical calculations on numbers.

Some of these operators include:

  1. Add: +
  2. Subtract: -
  3. Multiply: *
  4. Divide: /
  5. Remainder: %

The remainder operator is sometimes called a modulo, that returns the number that remains after the right-hand number divides into the left hand number as many times as it evenly can. For example, 11 % 3 equals 2, because 3 fits into 11 three times, leaving 2 as the remainder.

When logging a mathematic operation, do not use quotes. Example: console.log(20 + 1); will return 21. console.log('20 + 1'); will return 20 + 1.


String Concatenation

Operators can be used for other data types besides numbers too. When a + operator is used on two strings, it appends the right string to the left string. Example: console.log('I love ' + 'to code'); will print, I love to code.

The process of appending one string to another is called concatenation.

A computer will join two strings exactly, so if you want a space between two strings, you must manually add it in.


Properties

Every string has a property called length, that store the number of characters in that string. You can retrieve property information by appending the string with a period and the property name. Example: console.log('Hello'.length); will print 5.

The dot before a property is also an operator, called the dot operator.


Methods

Methods are actions that we can perform.

We call (use) these methods by appending an instance with the dot operator, the name of the method, and the opening and closing of parentheses. Example: 'example string'.methodName();

When we use console.log(), we're calling on the .log() method on the console object. Some examples of string methods are: console.log('hello'.toUpperCase()); prints 'HELLO', and console.log('Hey'.startsWith('H')); prints true.

Built-in Objects

In addition to console, there are other objects built in to JavaScript.

If you wanted to perform more complex mathematical operations than arithmetic, JavaScript has the built-in Math object.

Objects have methods. The Math object for example, has the .random() object: console.log(Math.random()); will print a random number between 0 and 1.

If you wanted to generate a random number between a different set of numbers, you simply multiply by that number. Example: console.log(Math.random() * 100); will print a random number between 0 and 100.

Additionally, if you wanted that random number to be a whole number, you could use the Math.floor() method, which takes a decimal number and rounds down to the nearest whole number. Example: console.log(Math.floor(Math.random() * 100)); will print a whole number between 0 and 100.

Section 2 Notes

Variables

A variable is a container for a value. Variables can be thought of as containers for information that lives in a computer's memory.

Information that can be stored in variables includes usernames, account numbers, or personalized greetings.

Variables are also useful for labeling data with a descriptive name, so that programs can be easier understood.

There are only a few things you can do with variables:

  1. Create a variable with a descriptive name.
  2. Store or update information stored in a variable.
  3. Reference or "get" information stored in a variable.

Variables are not the same thing as values. Variables contain values and represent them with a name.


Create a Variable: var

The var keyword declares a variable. Variables generally look as such: var varName = 'Name'; The variable can be called upon by inputting the variable name, in which it will print the value that the variable has stored.
Example:
var varName = 'Name';
console.log(varName);
will print a value of Name.

Camel casing is a standard convention in JavaScript for how to capitalize. You group all the words into one, and write the first word as lowercase, and the rest of the following words starting with uppercase. Example: thisIsTheVarName

The = symbol is the assignment operator, which assigns a value to a variable.

To print the variables value, simply use the variable name. Example: console.log(varName);

The rules for naming variables:

  1. Variable names will never start with a number
  2. Variable names are case sensitive
  3. Variable names cannot be the same as keywords

Create a Variable: let

The let keyword signals that the variable can be reassigned to a different value.

A variable can be declared even if it has no value, it will simply print undefined until it has been defined.

When using let, you can assign the variable a value at any time by simply typing the name of the variable, and setting it equal to a value. Example:
let varName = true;
// This will print true
varName = false;
// Now it will print false


Create a Variable: const

const is short for constant, and can be used to store a value just like var and let.

A const variable cannot be reassigned a value, because it is constant, and must be assigned a value when declared.

When deciding whether to use let or const, think about if you might need to reassign the value at a later point, if you won't need to, use const.


Mathematical Assignment Operators

Consider the example below:
let w = 4;
w = w + 1;
console.log(w);
// Output: 5

In the example, the variable w is assigned a value of 4, then is reassigned a value of w + 1, making the new value of w, 5. A different way of accomplishing this task would be to use mathematical assignment operators. We could rewrite the code above to say:
let w = 4;
w += 1;
console.log(w);
// Output: 5

In the second example, we use the += assignment operator to reassign the variable w. We are essentially performing the mathematical operation of the first operator (+) using the number to the right, then reassigning w to the computed value.

Other mathematical operators operate in a similar fashion: -=, *=, /=.


The Increment and Decrement Operator

The Increment operator is ++, and the Decrement operator is --.

The increment operator will increase the value of a variable by 1, and the decrement operator will increase the value of a variable by 1.

Example of how to use these two operators:
let increaseMe = 1;
let decreaseMe = 2;

increaseMe++;
decreaseMe--;
// Both variables now have a value of 1


String Concatenation with Variables

The + operator can be used to combine two string values even if those values are stored in variables.

Example:
let petName = 'Snoop';
console.log('My pets name is' + 'petName' + '.');
// This will print: My pets name is Snoop.


String Interpolation

In JavaScript, we can insert, or interpolate variables into strings using template literals.

Consider this example:
const myPet = 'sloth';
console.log(`I own a pet ${myPet}.`);
// This will print: I own a pet sloth.

A template literal is wrapped in backticks (`)

Template literals usually look like ${}


typeof Operator

If you need to check the data type of a variable's value, you can use the typeof operator.

The typeof operator checks the value to its right and returns a string of the data type. Example:
const unknown1 = 'hello';
console.log(typeof unknown1);
// The output would be: string

Conditional Statements

What are Conditional Statements?

A conditional statement checks specific conditions and performs a task based on those conditions.


The if Keyword

We can perform a task based on a condition using an if statement. Example:
if (true) {
console.log('This message will print!');
}
// Prints: This message will print!

An if statement is composed of:

  • The if keyword, followed by a set of parentheses which is followed by a code block indicated by a set of curley braces {}.
  • If the condition evaluates to true, the code inside the curly braces executes.
  • If the condition evaluates to false, the block won't execute.

If...Else Statements

If we wanted to add a default behavior for when a condition is false, we can use the else keyword to execute a code block for when the condition evaluates to false.

An else statement must be paired with an if statement, and operates in a similar fashion as the if statement, although it does not have a condition in parentheses.


Comparison Operators

Comparison operators are used for comparing different values, and include:

  • Less than: <
  • Greater than: >
  • Less than or equal to: <=
  • Greater than or equal to: >=
  • Is equal to: ===
  • Is NOT equal to: !==

Comparison operators compare the left value to the right value.

All comparison statements evaluate to either true or false, and are composed of:

  • Two values that will be compared
  • An operator that separates the values and compares them accordingly.

Logical Operators

Working with conditionals means that we will be using booleans

There are operators that work with boolean values, which are known as logical operators. Logical operators are used to add more sophisticated logic to our conditionals. The three logical operators are:

  • The and operator (&&)
  • The or operator (||)
  • The not operator, or the bang operator (!)

When we use the && operator, we are checking that two things are true. Example:
if (stoplight === 'green' && pedestrians === 0) {
console.log('Go!');
} else {
console.log('Stop');
}

When using the && operator, both conditions must evaluate to true for the entire condition to evaluate to true and execute.

If we only care about either condition being true, we can use the || operator. Example:
if (day === 'Saturday' || day === 'Sunday') {
console.log('Enjoy the weekend!');
} else {
console.log('Do some work.');
}

When using the || operator, only one of the conditions must evaluate to true for the overall statement to evaluate to true.

Logical Operators

Logical operators use boolean values (true and false).

The three types of logical operators are:

  1. The and operator (&&)
  2. The or operator (||)
  3. The not operator (!)

When using the && operator, both conditions must evaluate to true for the code to execute. Example:
if (stoplight === 'green' && pedestrians === 0) {
console.log('Go!');
} else {
console.log('Stop.');
}

If only one of two or more conditions needs to evaluate to true, we use the || operator. If both of the conditions are false, the else code will execute.

Example:
if (day === 'Saturday' || day === 'Sunday') {
console.log('Enjoy the weekend!');
} else {
console.log('Do some work.');
}

The ! operator reverses the value of a boolean.

Example:
let sleepy = true;
console.log(!sleepy);
// Prints false



Truthy and Falsy

We might need to check if a variable exists, without needing to check its actual value.

Examples of falsy values include: 0, empty strings ('' or ""), null, undefined, and NaN (Not a Number).

To check if a variable is truthy or falsy, make the condition of an if statement that variable. Example:
let myVariable = 0;
if (myVariable) {
console.log('The variable has a value');
} else {
console.log('The variable does not have a value');
}
// Prints: The variable does not have a value



Truthy and Falsy Assignment

You can use truthy and falsy evaluations to assign variables.

With boolean conditions, JavaScript assigns a truthy value to a variable if you use the || operator in the assignment. Example:
let defaultName = username || 'Stranger';

The || operator checks the left hand condition first, so the variable defaultName will be assigned the actual value of username if it is truthy, and 'Stranger' if username is falsy. This is referred to as short circuit evaluation



Ternary Operator

Ternary Operators shorten if...else statements. To use a ternary operator, write the variable, followed by a ?, then write the code to be executed if the statement evaluates to true, seperate the else statement with a colon, and don't forget the semicolon.

isNightTime ?
console.log('Turn on the lights!') : console.log('Turn off the lights!');



Else If Statements

You can add as many else if statements as you like.

The else if comes after the if statement, and before the else statement.

if...else if...else statements are read from top to bottom, so the first condition that evaluates to true will be executed.



The Switch Keyword

We can use the switch keyword if we want to check a lot of conditions.

A switch statement consists of the switch keyword, followed by a condition in parentheses, then "cases" that contain code to be executed. Example:
switch (groceryItem) {
case 'tomato':
console.log('Tomatoes are $0.49');
break;
case 'lime':
console.log('Limes are $1.49');
break;
default:
console.log('Invalid Item');
break;
}

The value that is inside the parentheses following the switch keyword contains the value that each case will compare to.

Inside the block {} there are multiple cases that check if the expression matches the specified value that comes after it.

The break keyword tell the computer to exit the block and not execute anymore code or check any other cases inside the code block. Without the break keyword, a switch will execute the code for all matching cases.

There should always be a default statement at the end of a switch. This tells the computer what to do if none of the cases are true.
