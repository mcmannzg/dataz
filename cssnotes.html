CSS Notes

This page is to act as a place to store all of the notes I have taken on CSS directly from codecademy.

Positioning:

There are four types of position values you can specify:

position: static; is the default

position: relative; gives the element a position relative to its static position on the webpage. If you use relative position, you must follow it with top (moves the element down), bottom (moves the element up), left (moves the element right), or right (moves the element left).

position: absolute; If you use absolute position, all other elements on the page will ignore the element as if it is not on the page.

position: fixed; Fixes an element to the page, regardless of scrolling.



Z-Index

Can be thought of as depth of elements on a page.

The higher the z-index, the more an element is positioned towards the top.

Z-index has no effect on elements with position: static;



Display

There are three values you can use with display:

display: inline; Causes elements to only take up the amount of space needed to display their content, and cannot be modified with height and width. Elements with the inline property will be displayed alongside other elements with the inline property.

display: block; Elements with the block property fill the entire width of the page by default. Unless otherwise specified, they are the height necessary to accommodate their content.

display: inline-block; Content inside a div with the inline-block property will be displayed side by side.



Float

Moves the element as far right or left on the page.

You can float multiple elements at once.



Clear

Specifies how elements should behave when they bump into eachother on the page.

Can be specified to: left, which means the left side of the element will not touch any other element within the same containing element. right, the right side of the element will not touch any other element within the same containing element. both, neither side of the element will touch any other element within the same containing element. none, the element can touch either side.
